<PlanBadge tier="enterprise" />

Enterprise is for organisations with compliance, scale, or isolation requirements
that go beyond what Pro covers. It's everything in Pro plus:

- **SSO / SAML** integration with Okta, Azure AD, Google Workspace.
  <PlanBadge tier="enterprise" planned />
- **RBAC** — role-based access control over dashboard actions, policies, channels.
  <PlanBadge tier="enterprise" planned />
- **SIEM integration** — native exporters for Splunk, Elastic, Sentinel, Datadog,
  or any HTTP-based SIEM.
- **Compliance reporting** — SOC 2 / ISO 27001 / GDPR / HIPAA-shaped audit
  bundles, exportable on schedule.
- **Air-gapped deployment** — offline license activation, self-hosted dashboard,
  manual policy distribution. <PlanBadge tier="enterprise" planned />
- **Custom security filters** — operator-authored filters shipped via the
  central policy pipeline.
- **Multi-team scope** — one Enterprise account, many teams, federated billing.
- **SLA + dedicated support** — named contact, response-time guarantees.
- **Unlimited users**.

## Migration from Pro

Existing Pro teams can upgrade to Enterprise in the dashboard with no downtime
or data migration. Policies, audit summaries, and team keys are preserved;
Enterprise-only features unlock.
